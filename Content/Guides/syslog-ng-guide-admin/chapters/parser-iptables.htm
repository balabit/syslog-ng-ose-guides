iptables parser

The iptables parser can parse the log messages of the iptables command. Available in version 7.0.93.16 and later.

Declaration:
@version: 
@include "scl.conf"
log {
    source { system(); };
    parser { iptables-parser(); };
    destination { ... };
};

The iptables-parser() is actually a reusable configuration snippet configured to parse iptables messages. For details on using or writing such configuration snippets, see more. You can find the source of this configuration snippet on GitHub.

By default, iptables-parser() uses the iptables. prefix. To modify it, use the following format:

parser { iptables-parser(prefix("myprefix.")); };
