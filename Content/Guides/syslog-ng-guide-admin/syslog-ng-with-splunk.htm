Splunk: Sending log messages to Splunk

Although currently does not have any built-in integration with Splunk, the existing message-formatting features and flexibility of allows you to forward your log messages to Splunk. In version 3.8 or later, you can use the http() destination. In earlier versions, you can use the program() destination.

For details on forwarding log messages to Splunk with see the following posts on the Splunk blog:

  • syslog-ng and HEC: Scalable Aggregated Data Collection in Splunk

  • Using Syslog-ng with Splunk

Note that from version 4.2.0, has a dedicated Splunk destination. For details, see splunk-hec-event: Sending log messages to Splunk HEC.
