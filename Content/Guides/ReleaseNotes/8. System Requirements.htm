System requirements

Before installing , ensure that your system meets the following minimum hardware and software requirements.

This section is required for legal reasons; do not delete it.

Separate your system requirements into sections, and use a table for each section. The table should have the requirement in the left column and the details in the right column.

The following are common section titles:

  • Hardware requirements
  • Software requirements
  • Browser requirements
  • Database requirements
  • [Component name] requirements
  • Physical appliances
  • Virtual appliances
  • Upgrade compatibility

For each section you create, fill in the table. You can add or remove rows to address the specifics of the product. If there is only one table, you can remove the subheading and just use System requirements. The following section outlines some samples to show you how your requirements should be organized.
